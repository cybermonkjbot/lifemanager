# Support

## Where to Ask for Help

- Questions and usage help: open a GitHub issue.
- Bug reports: use the bug report issue template.
- Feature requests: use the feature request issue template.
- Security issues: follow the process in [SECURITY.md](./SECURITY.md).

## What to Include

To get help quickly, include:
- what you expected to happen
- what happened instead
- steps to reproduce
- environment details (OS, Bun, Node, browser)
- relevant logs/screenshots with secrets removed

## Scope and SLA

This project is maintained on a best-effort basis.
Response times can vary.
